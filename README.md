# udp-kernel-path-bench

Linux 환경에서 UDP echo server와 load generator를 이용해 패킷 송수신 경로의 성능을 관찰하기 위한 벤치마크 프로젝트입니다.

이 프로젝트는 애플리케이션 수준의 복잡한 처리 없이, UDP 패킷을 받고 다시 보내는 단순한 echo workload를 사용해 운영체제의 네트워크 처리 경로에 집중할 수 있도록 구성되었습니다. 이를 통해 UDP 송수신 과정에서 발생하는 지연, 처리량 변화, 패킷 손실, 그리고 커널 경로의 병목 가능성을 실험적으로 확인하는 것을 목표로 합니다.

서버는 하나의 UDP 포트에서 여러 클라이언트의 요청을 받아 그대로 되돌려주는 구조로 동작하며, 부하기는 여러 프로세스를 통해 동시에 UDP 패킷을 전송하여 서버에 일정한 부하를 가합니다. 이를 바탕으로 단순 기능 확인부터 기초적인 성능 비교 실험까지 확장할 수 있습니다.
